Running the autotester in Release Mode:
1. Go to folder source\Static Program Analyzer\Release and open the command prompt
2. Add the source file and test cases that you want to test into the folder
3. Run the command in the command prompt:
	AutoTester	SourceFile.txt 	TestCaseFile.txt outputFile.xml
